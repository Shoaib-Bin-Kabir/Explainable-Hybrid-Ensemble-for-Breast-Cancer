# Explainable Hybrid Ensemble for Breast Cancer

## Project Overview

This project develops a high-performance hybrid ensemble model for breast cancer prediction with strong emphasis on explainability and validation.

## Main Focus

- Ensemble-based classification
- Evaluation through multiple performance metrics
- Explainable AI integration for model transparency

## Repository Structure

- Project Code.ipynb: End-to-end notebook for preprocessing, training, evaluation, and explainability
- Project_Details.txt: Detailed project methodology and workflow explanation

## Notes

Project_Details.txt explains the full stepwise pipeline and model interpretation strategy.
